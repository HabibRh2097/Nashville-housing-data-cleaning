#  Nashville Housing Data Cleaning & Visualization

###  Project Overview
This project focuses on cleaning and analyzing Nashville housing data using **SQL** and **Power BI**.  
The goal was to fix missing data, standardize fields, remove duplicates, and visualize key trends in property pricing.
